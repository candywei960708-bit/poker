# 德州撲克

- **系級班級** : 資工1A
- **組別** : 第15組
- **組長** : 郭育宏
- **組員** : 張舜凱、吳宥均
  
## 遊戲規則
### 遊戲基礎與目標

德州撲克是一種使用 52張撲克牌的公共牌遊戲。

目標： 在遊戲結束時，結合自己的2張底牌和桌面上5張公共牌，組成最強的5張牌組合，或是透過下注讓對手蓋牌來贏得底池。

### 遊戲流程

1. 遊戲初始化 
建立玩家：程式會先加入一名玩家 "You" 並根據輸入數量加入 NPC。

準備牌組：系統會生成 52 張牌並進行洗牌。

設定籌碼：每位玩家預設擁有 1000 籌碼。

2. 單回合流程 
每一輪遊戲的標準步驟如下：

第一步：下注與發底牌 
強制底注：每位玩家自動扣除 10 籌碼進入底池。

發放底牌：每人分發 2 張只有自己能看見的牌。

狀態更新：玩家可以在畫面看到自己的手牌。

第二步：玩家決策 
選擇行動：玩家需輸入指令選擇 加注 、過牌  或 蓋牌 。

NPC 跟進：如果有玩家加注，其他 NPC 也會跟進下注 20 籌碼進入底池。

第三步：發布公共牌 
翻開公牌：系統從牌組中取出 5 張公共牌展示在桌面上。

統計底池：畫面會顯示當前累積的總獎金。

3. 結算與勝負 
計算分數：Evaluator會結合玩家手牌與公共牌，根據牌型計算分數。

一對 ：得 100 分。

三條 ：得 400 分。

判定贏家：分數最高且未蓋牌的玩家為贏家。

分配獎金：贏家獲得底池內的所有籌碼。

4. 回合結束 
顯示狀態：系統列出所有玩家剩餘的籌碼量。

詢問續玩：詢問玩家是否繼續下一輪，若選擇 'y' 則清空手牌並重新開始。
